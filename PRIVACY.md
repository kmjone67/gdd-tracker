# Privacy Policy

This GPT Action is used to read a public Bermuda grass Growing Degree Day log generated from historical weather data for Princeton, Texas.

## Data Accessed

The Action retrieves a public JSON file containing weather-derived lawn care data, including dates, daily high temperatures, daily low temperatures, base temperature, and calculated GDD values.

The Action does not collect, store, sell, or share personal information from users.

## User Data

No user-provided personal data is sent to or stored by this Action.

## Third-Party Services

The GDD log is hosted on GitHub. The weather data used to generate the log is retrieved separately by an automated GitHub Actions workflow from WeatherAPI. The GPT Action itself only reads the already-generated public JSON log.

## Contact

For questions about this privacy policy, contact the repository owner.
